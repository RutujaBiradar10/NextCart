# 🛒 NextCart – Online Shopping Web Application

NextCart is a modern and responsive e-commerce web application built using HTML, CSS, and JavaScript. It provides a smooth and interactive online shopping experience with features like product browsing, search, filtering, cart management, and order tracking.

---

## 🚀 Features

- 🏠 Home page with categories and hero section  
- 🔍 Product search functionality  
- 🏷️ Category-based product browsing  
- 🎯 Filter products by price, brand, and rating  
- 🛒 Add to cart and manage quantities  
- 📦 Order placement and order history  
- 👤 User account page for personal details  
- 📱 Fully responsive design (mobile + desktop)  
- 📂 Sidebar navigation for mobile users  

---

## 🛠️ Technologies Used

- HTML5 – Structure  
- CSS3 – Styling and responsive design  
- JavaScript – Dynamic functionality
